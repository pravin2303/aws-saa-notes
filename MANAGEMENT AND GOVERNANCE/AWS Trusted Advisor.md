## AWS Trusted Advisor – CLF-C02 Scope 🧠📊

**AWS Trusted Advisor** provides **real-time guidance** to help you follow AWS best practices.

It analyzes your AWS environment and gives recommendations in key areas.

---

## What Trusted Advisor Checks

It provides recommendations in five main categories:

1️⃣ **Cost Optimization** 💰  
2️⃣ **Performance** 🚀  
3️⃣ **Security** 🔐  
4️⃣ **Fault Tolerance** 🛡️  
5️⃣ **Service Limits** 📈

---

## Example Recommendations

- Unused EC2 instances → Save cost
    
- S3 bucket with public access → Security risk
    
- Underutilized load balancer → Optimization suggestion
    

---

## Why It Is Important

✔ Helps reduce AWS costs  
✔ Improves security  
✔ Enhances reliability  
✔ Optimizes performance

---

## Trusted Advisor vs CloudWatch (Exam Concept)

|Service|Purpose|
|---|---|
|CloudWatch|Monitor metrics & set alarms|
|Trusted Advisor|Best practice recommendations|

---

## Example (Exam Scenario)

If question says:

“Company wants recommendations to reduce AWS costs.”

Correct answer → AWS Trusted Advisor.
