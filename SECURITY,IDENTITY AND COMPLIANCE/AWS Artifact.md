# 🧠 AWS Artifact

## 🔗 Service Type
#Compliance #Governance #Audit #Documentation

---

## 📌 What is AWS Artifact?

AWS Artifact is a service that:

- Provides access to AWS compliance reports
- Allows you to review and download audit documents

👉 Artifact = “Compliance documentation center”

---

## ⚙️ How It Works (Architecture Thinking)

1. User logs into AWS

2. Access AWS Artifact

3. Download:
   - Compliance reports
   - Agreements

👉 Access → Review → Download
